# 🏦 Python ATM System

A simple ATM simulation built using Python. This project demonstrates Object-Oriented Programming (OOP) concepts.

## Features
- PIN verification
- Balance check
- Withdraw money
- Deposit money
- Account number masking

## Technologies Used
- Python
- OOP
- Loops and conditional statements

## How to Run

1. Clone the repository

git clone https://github.com/yourusername/python-ATM-system.git

2. Run the program

python atm.py
